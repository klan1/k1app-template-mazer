# k1app-template-mazer

Free and Open-source Bootstrap 5 Admin Dashboard Template and Landing Page

## Requirements
- K1.lib v2.0+
- PHP 8.1 

## Installation

```sh
composer require klan1/k1app-template-mazer
```
